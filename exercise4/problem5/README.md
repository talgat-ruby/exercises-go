# Problem 5

`send` combines words into a single message. Your job with minimum changes implement correct behaviour.

```go
list := []string{
    "Hello",
    "dear",
    "friend!",
    "Learn",
    "from",
    "yesterday.",
    "Save",
    "our",
    "soles.",
}

send(list) // "Hello dear friend! Learn from yesterday. Save our soles."
```
